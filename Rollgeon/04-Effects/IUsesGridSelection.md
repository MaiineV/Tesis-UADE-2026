# IUsesGridSelection

> Marker interface declaring that the effect uses a boolean grid
> selection (`bool[,]` rendered with Odin's `[TableMatrix]`). The
> inspector shows the matrix when this capability is present.

## Dependencies
**Used by:** AoE-shape effects authored as a fixed grid pattern.

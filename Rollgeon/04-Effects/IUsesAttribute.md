# IUsesAttribute

> Marker interface declaring that the effect reads an attribute from a
> source entity (self / opponent / triggering). Implementing this
> tells the inspector to reveal the attribute-source dropdown.

## Dependencies
**Used by:** effect inspectors (Odin); capability discovery.
